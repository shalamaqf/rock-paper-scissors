## Title
Rock, Paper, Scissors.

## Description
This is a simple game project based on Rock-Paper-Scissors. The game is played between two players: a human and the computer. In each round, both players choose one of the three options â€” rock, paper, or scissors. The winner of the round is determined based on the standard rules:

- Rock beats scissors

- Scissors beats paper

- Paper beats rock

The first player to win 5 rounds is declared the overall winner.
